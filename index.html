Tables

Tags

Tag Name Description
<table> Table This is the wrapper element for all HTML tables.
<thead> Table head This tag is used to group one or more <th> elements into a header for a table. The <thead> element must be used within a <table> element, and it typically appears before the first <tbody> element. It gives a clear visual distinction between the table header and the table body.
<th> Table header This tag is used to define a header cell in a table. It must be used within a <tr> element as a child element to define the header row. By using the <th> element, you can create a row in a table that stands out and visually separates the header information from the main data.
<tbody> Table body This tag is used to group one or more <tr> elements into the main body of a table. The <tbody> element must be used within a <table> element, and it typically appears after the <thead> element. It gives a clear visual distinction between the table header and the table body.
<tfoot> Table head This tag is used to group one or more <tr> elements into the footer for a table. The <tfoot> element must be used within a <table> element, and it typically appears after the last <tbody> element. It gives a clear visual distinction between the table body and the table footer. This can be usefol for displaying summary information or totals at the bottom of a table.
<tr> Table row This element must be used within a <table> element and it typically contains one or more <td> or <th> elements to define the cells in the row. By using the <tr> element, you can create a structured layout for tabular data, which can be styled and manipulated using CSS and JavaScript.
<td> Table data This tag is used to define table data in an HTML table. The element must be used within a <tr> (table row) element to define the cells in a row. The conent of the cell can include text, images, links, or other HTML elements.
<caption> Caption A table caption is a description that provides context and summary information for a table. It usually appears at the top of the table. It must be written before <thead>.
<colgroup> Column group This tag is used to define a group of columns in a table. It allows you to apply formatting and styling to a group of columns at once, rather than having to apply the styling to each individual column separately.
<col> Column This tag defines a single column within a table. It must be used within a <colgroup> element, and it's typically used to apply styling or formatting to a group of columns within a table.

Attributes

Tag Name Description
colspan Column span colspan is an attribute that can be applied to a <td> or <th> element to specify how many columns the cells should span. This allows you to merge adjacent cells in a row to create a larger cell that spans multiple columns.
rowspan Row span Rowspan is an attribute that can be applied to a <td> or <th> element to specify how many rows the cells should span. This allows you to merge adjacent cells in a row to create a larger cell that spans multiple rows.
border Border Specifies the border around the table and its cells.
cellpadding Cellpadding Specifies the amount of space between the cell content and the cell wall.
width Width Specifies the width of the table in pixels or as a percentage of the available space.
height Height Specifies the height of the table in pixels or as a percentage of the available space.
align Align Specifies the horizontal alignment of the table within the surrounding context. This attribute is deprecated in HTML5 and it is recommended to use CSS instead.
bgcolor bgcolor Specifies the horizontal alignment of the table within the surrounding context. This attribute is deprecated in HTML5 and it is recommended to use CSS instead.
